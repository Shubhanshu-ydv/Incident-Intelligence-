# Screenshots Directory

This directory is for storing visual assets for the README.

## Required Screenshots:

1. **dashboard.png** - Main dashboard view showing incidents list
2. **ai-chat.png** - AI chat interface with example query/response
3. **flow-diagram.png** - Data flow visualization from /flow route
4. **live-updates.gif** - Short GIF showing real-time update (optional but impressive)

## How to Capture:

1. Run the application (all services)
2. Create 3-5 sample incidents
3. Use screenshot tool (Windows: Win+Shift+S)
4. For GIF: Use tools like ScreenToGif or LICEcap
5. Save files with exact names above

## Tips:

- Use high contrast/visibility settings
- Show meaningful data (not placeholder text)
- Crop images to relevant content only
- Keep file sizes reasonable (<500KB per image)
